# 欢迎来到老中传习录！



## 为什么要建立这个网站？

其实以前一直都很想建一个个人网站，但因为众所周知的原因在国内一般很难做到，再加上自己不太会写html，所以一直没搞成🤔。但是现在网络环境的变化和AI的发展，导致客观条件成熟了。加上研究室的同学一直在拿AI做很多有趣的事情，所以干脆我也来做一个吧！这个网站主要收录我个人或别人的各种经验贴，以及一些有趣的小玩意儿。

另外一个很重要的原因就是帮助加入研究室的新人快速上手深度学习。从2022年chatGPT横空出世深度学习爆火后，包括笔者在内的许多人都想要学习深度学习，然而作为一个实际上才扩张了没几年的学科，前人的资料严重不足，几乎所有人都在各自摸索。更变态的是这一学科当前的发展特点就是学习速度远远跟不上行业发展速度，经常是刚学会某项技能，他就落后了，以至于不得不紧跟最新论文来硬啃学习，开局难度曲线极大。举个例子，像当年秒杀一众棋圣的阿尔法狗，在2016年是如此地强大与惊艳，然而到了2022年左右，其主要原理CNN早就变成深度学习里的一个课后作业了，更不用说现在又过了几年，当年的GPT2和DALLE2都能拿来当大作业用。刚出新手村就遇上Lv.100的BOSS，瞬间去世，实在令人难受。

最后就是一个现实原因，笔者这一年即将毕业，而我的学长学姐们也恰好今年同时毕业，一下子导致研究室没人老带新，因此抓紧机会赶紧把大家的经验尽量写下来，这样，未来新加入研究室的同学都能很快上手。

因此本专栏的特点就是手把手教，假定你是一个0基础小白，仅仅接触过编程语言如C++、Python或数据结构的程度（不会吧，都8102年了还有理工科学校不教这些吗）。哪怕你真的连这些都没学过，这里也会尽量提供对应教程，帮助你半年内快速上手，你只需要会使用计算机的基础操作就行。主打一个从入门到入土。

希望本专栏能对你深度学习求学之旅有用！

## 📖神人师父

###### 其实就是深度学习教程啦

收集笔者遇到过甚至学过的，很不错的国内外深度学习教程。要知道，深度学习基本上不太可能在大学课堂或硕士课程里得到好好学习的。我想看过国内的上交自救指南的人都深知，国内本科教育可以说是烂完了，在这里分清北烂一点还是985211烂一点是毫无意义的，所以一上大学就要做好全面自习的准备。如果你在国外，除非在美国，不然大概率也是要自习的。

该栏目暂定分为以下几个板块：

* 数学基础：包括微积分、微分方程等必备课程
* 编程语言：主要为Python，附带讲解Git或C++之类的语言，以及深度学习需要用到的Pytorch等
* 深度学习：正式进入深度学习的学习，追踪主流模型原理
* 计算机架构：主要关注硬件方面，比如CPU是如何构造的，RISCV架构等
