# OI-Wiki

[在线直达 → oi-wiki.org](https://oi-wiki.org/)

## 简介

OI-Wiki 是一个免费、开放、持续更新的竞赛编程知识库。"OI" 即信息学奥林匹克竞赛（Olympiad in Informatics）。它把 OI / ICPC 所需的算法与数据结构知识系统地整理在一起，从入门到高级全覆盖，是国内竞赛选手最常用的参考站点之一。

## 内容覆盖

| 方向 | 代表主题 |
|------|---------|
| 基础 | C++ 语言、复杂度分析、STL |
| 数据结构 | 线段树、树状数组、并查集、Link-Cut Tree |
| 图论 | 最短路、最小生成树、网络流、二分图 |
| 动态规划 | 背包、区间 DP、树形 DP、状压 DP |
| 字符串 | KMP、AC 自动机、后缀数组、SAM |
| 数学 | 数论、组合数学、多项式、博弈论 |
| 计算几何 | 凸包、半平面交、旋转卡壳 |
| 实用 | 评测环境、输入输出优化、竞赛技巧 |

## 适合人群

- 备战 NOI / NOIP / CSP 的中学生
- 参加 ICPC / CCPC 的大学生
- 刷力扣遇到图论、DP 不会的普通人

- **开源**：[GitHub 仓库](https://github.com/OI-wiki/OI-wiki) 接受社区贡献，内容持续迭代
- **结构清晰**：按知识点分层，循序渐进，不会一上来就扔公式
- **免费无广告**：纯公益性质，CC BY-SA 4.0 许可
