斗地主 · 游戏规则

基本设置

3 人游戏，1 副 54 张牌（含大小王）。1 人为地主，2 人为农民。

叫地主

从随机玩家开始，每人依次叫分（1-3 分）或不叫（0 分），叫分最高者得底牌（3 张）成地主。若无人叫牌则重新发牌。

出牌规则

地主先出，顺时针轮流。出牌须同类型且比上家大，或选择不出（过）。所有其他人都不出时，最后出牌者再次领出。

合法牌型

  • 单张、对子、三张 — 基本牌型
  • 三带一 / 三带二 — 三张 + 1单张 / + 1对子
  • 顺子 — 5张及以上连续单张（3-A，不含 2 和王）
  • 连对 — 3对及以上连续对子（不含 2 和王）
  • 飞机 — 2组及以上连续三张（可带单或带对）
  • 四带二 — 4张同点 + 2张（单或对均可）
  • 炸弹 — 4张同点，可压任意非火箭牌型
  • 火箭 — 大王 + 小王，最强，无法被压

牌力：3 < 4 < … < K < A < 2 < 小王 < 大王

胜负计分

先出完手牌者一方获胜。底分 = 叫分，每张炸弹/火箭翻倍一次。

快捷键

Enter / 空格 = 出牌  |  P = 不出  |  H = 提示
