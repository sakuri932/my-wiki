# 证件照处理工具

<style>
  .md-grid { max-width: 100% !important; }
  .md-content { width: 100% !important; }
</style>

一个纯本地运行的证件照处理工具，所有操作在浏览器内完成，不上传任何数据。

主要功能：
- 🔄 点击背景区域一键换底色（白、蓝、红等常用证件照背景）
- 📐 多种标准证件照尺寸裁切（一寸、二寸、护照等）
- 🎭 表情包编辑器
- 🖨️ 一键生成打印排版

也可以单独打开一个[网页](../index.html)，这样更完整。
